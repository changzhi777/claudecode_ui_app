# 斜杠命令自动补全功能 🎯

## 功能特点

### ✨ Tab 键自动补全
- 输入 `/` 后自动显示命令列表
- 按 `Tab` 键手动触发补全
- 支持键盘导航（↑↓ 选择，Enter 确认，Esc 取消）

### 📋 支持的命令分类

#### 通用
- `/help` - 显示帮助信息
- `/?` - 显示帮助信息

#### 对话
- `/clear` - 清空当前对话
- `/save` - 保存当前对话
- `/new` - 新建对话

#### 系统
- `/exit` - 退出应用

#### 设置
- `/model` - 切换 AI 模型
- `/theme` - 切换主题

#### 信息
- `/token` - 查看 token 使用情况

## 使用方法

### 1. 自动触发
输入框中输入 `/` 即可自动显示命令列表

### 2. 手动触发
在任何位置按 `Tab` 键触发补全

### 3. 键盘导航
- **↑↓** - 上下选择命令
- **Enter** - 确认选择
- **Esc** - 取消补全

### 4. 鼠标操作
- 点击命令即可选择
- 点击外部区域关闭补全

## 特性

- 🚀 **实时过滤** - 输入命令时自动过滤匹配项
- 🎨 **分类显示** - 按类别组织命令，更易查找
- ⌨️ **快捷键支持** - 完整的键盘导航
- 🎯 **智能定位** - 自动定位光标到合适位置
- 🌈 **视觉反馈** - 选中项高亮显示

## 示例

```
用户: /he[Tab]
显示: /help, /?

用户: /cl[Tab]
显示: /clear

用户: /model[Enter]
执行: /model 命令
```
