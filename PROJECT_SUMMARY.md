## 🎯 项目目标

为 ClaudeCode CLI 打造现代化的桌面 UI 界面，提供类似 ChatGPT 的流畅对话体验，同时集成强大的代码编辑与文件管理能力。

## 📊 技术实现

### 架构设计

```
┌─────────────────────────────────────────┐
│         ClaudeCode UI App               │
├─────────────────────────────────────────┤
│  Electron 主进程                         │
│  ├── CLI 进程池管理                      │
│  ├── IPC 通信处理器                      │
│  └── 配置读取服务                        │
├─────────────────────────────────────────┤
│  Worker 线程                             │
│  └── CLI 业务逻辑处理                    │
├─────────────────────────────────────────┤
│  React 渲染进程                          │
│  ├── 对话界面 (ChatUI)                   │
│  ├── 代码编辑器 (Monaco)                │
│  ├── 文件树 (FileTree)                  │
│  ├── 任务可视化 (TaskViz)               │
│  ├── 成本统计 (CostStats)               │
│  └── 设置面板 (Settings)                 │
├─────────────────────────────────────────┤
│  Zustand 状态管理                        │
│  ├── CLI Store                          │
│  └── Editor Store                       │
└─────────────────────────────────────────┘
```

### 核心技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| 桌面框架 | Electron 30.5 | 跨平台桌面应用 |
| 前端框架 | React 18.3 | UI 组件库 |
| 构建工具 | Vite 5.2 | 快速开发与热更新 |
| UI 组件 | shadcn/ui | 高质量可定制组件 |
| 状态管理 | Zustand 4.5 | 轻量级全局状态 |
| 代码编辑 | Monaco Editor | VS Code 同款编辑器 |
| 测试框架 | Vitest 1.6 | 单元测试与集成测试 |
