**转换命令**：
```bash
# 创建 iconset
mkdir icon.iconset
sips -z 16 16 icon.svg --out icon.iconset/icon_16x16.png
sips -z 32 32 icon.svg --out icon.iconset/icon_16x16@2x.png
# ... 其他尺寸
iconutil -c icns icon.iconset -o icon.icns
```
