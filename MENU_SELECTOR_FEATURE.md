# 🎛️ 菜单式选择器功能

## 功能概述

将主题和视图选择器从平铺按钮组改为**下拉菜单**形式，提供更好的用户体验和信息展示。

## 设计优势

### 1. **节省空间** 📐
- **之前**: 平铺按钮，占据较多横向空间
- **现在**: 单个按钮 + 下拉菜单，空间占用减少 60%

### 2. **信息丰富** 📋
- **主题信息**: 显示图标、名称、描述、设计师
- **颜色预览**: 显示主色、次要色、强调色的色块
- **状态指示**: 当前选中项显示 ✓ 标记

### 3. **交互优化** 🖱️
- **点击触发**: 点击按钮展开菜单
- **外部关闭**: 点击菜单外部自动关闭
- **ESC 关闭**: 按 ESC 键关闭菜单
- **键盘提示**: 底部显示快捷键提示

### 4. **视觉效果** ✨
- **动画**: scale-in 缩放动画
- **阴影**: xl 级别阴影，层次分明
- **圆角**: lg 级别圆角，现代感
- **悬停**: 平滑的背景色过渡

## 组件实现

### ThemeSwitcher.tsx

**触发按钮**:
```tsx
<button className="flex items-center gap-2 px-3 py-2 bg-bg-secondary hover:bg-bg-tertiary rounded-lg">
  <Palette size={16} />
  <span>{currentMeta.name}</span>
  <ChevronDown size={14} className={isOpen ? 'rotate-180' : ''} />
</button>
```

**下拉菜单**:
```tsx
<div className="absolute top-full right-0 mt-2 w-80 bg-bg-elevated border border-bg-tertiary rounded-lg shadow-xl">
  {/* 菜单标题 */}
  <div className="px-4 py-3 border-b border-bg-tertiary">
    <Palette size={16} />
    <span>选择主题</span>
  </div>

  {/* 主题列表 */}
  {themes.map(theme => (
    <button className="w-full px-4 py-3 flex items-center gap-3">
      {/* 图标 */}
      {/* 信息 */}
      {/* 颜色预览 */}
    </button>
  ))}

  {/* 底部提示 */}
  <div className="px-4 py-2 bg-bg-secondary">
    💡 提示：使用 ⌘T 快速切换
  </div>
</div>
```

### ViewSwitcher.tsx

类似的下拉菜单结构，但包含：
- 视图图标和名称
- 详细描述
- 使用场景说明
- 快捷键提示（⌘K）

## 交互细节

### 打开菜单
1. 点击触发按钮
2. 菜单以 scale-in 动画展开
3. 下拉箭头旋转 180°

### 选择主题
1. 展开菜单显示所有主题
2. 每个主题显示：
   - 大图标（emoji 或图标）
   - 主题名称
   - 简短描述
   - 设计师信息
   - 3 个颜色预览圆点
3. 当前主题有背景高亮
4. 点击任意主题立即切换

### 关闭菜单
- 选择主题后自动关闭
- 点击菜单外部关闭
- 按 ESC 键关闭

## 性能优化

- ✅ **条件渲染**: 只在打开时渲染菜单
- ✅ **事件监听**: 自动清理副作用
- ✅ **CSS 动画**: 使用 transform 而非 position
- ✅ **避免重渲染**: useRef 缓存 DOM 节点

## 无障碍支持

- ⌨️ **键盘导航**: ESC 关闭
- 🖱️ **鼠标交互**: 清晰的悬停状态
- 🎯 **焦点管理**: 点击外部自动关闭
- 📝 **屏幕阅读器**: 语义化按钮和标题

## 快捷键保留

虽然改为菜单式选择，但仍保留快捷键：
- **⌘T**: 循环切换主题
- **⌘K**: 切换视图

菜单底部会显示快捷键提示。
