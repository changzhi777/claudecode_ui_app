# 🎨 专业深色主题 - UI/UX 优化

## 设计理念

创建一个**专业、高效、接地气**的深色主题，专为长时间开发工作设计。

**核心理念**：
- ✅ 去除"AI 味"配色（无紫色、粉色、荧光色）
- ✅ 使用经典的深色开发工具配色
- ✅ 高对比度但不过度刺眼
- ✅ 专业的蓝色主色调
- ✅ 绿色成功提示（自然、积极）

## 配色方案

### 主色调
```css
/* 蓝色主色 - 专业、可信赖 */
--color-primary: #3B82F6

/* 深灰次要色 - 沉稳、不干扰 */
--color-secondary: #1F2937

/* 绿色强调色 - 成功、积极 */
--color-accent: #10B981
```

### 背景色
```css
/* 主背景 - 深黑，护眼 */
--bg-primary: #0A0E17

/* 次级背景 - 深灰，层次分明 */
--bg-secondary: #111827

/* 三级背景/边框 - 中灰，分割清晰 */
--bg-tertiary: #1F2937

/* 悬浮元素 - 一致的背景色 */
--bg-elevated: #1F2937
```

### 文本色
```css
/* 主要文本 - 高对比度，易读 */
--text-primary: #F9FAFB

/* 次要文本 - 柔和，不刺眼 */
--text-secondary: #D1D5DB

/* 三级文本 - 低对比度，提示性 */
--text-tertiary: #9CA3AF

/* 反色文本 - 用于深色背景 */
--text-inverse: #0A0E17
```

### 语义色
```css
/* 错误 - 红色（经典） */
--color-error: #EF4444

/* 成功 - 绿色（积极） */
--color-success: #10B981

/* 警告 - 黄色（醒目） */
--color-warning: #F59E0B

/* 信息 - 蓝色（与主色一致） */
--color-info: #3B82F6
```

### 组件配色
```css
/* 聊天界面 */
--chat-user-bubble: #3B82F6  /* 用户消息 - 蓝色 */
--chat-ai-bubble: #1F2937     /* AI 消息 - 深灰 */
--chat-input-bg: #1F2937      /* 输入框 - 深灰 */

/* 代码编辑器 */
--editor-bg: #0A0E17          /* 编辑器 - 主背景 */
--editor-gutter: #111827       /* 行号栏 - 次级背景 */
--editor-line-highlight: #1F2937  /* 当前行 - 高亮 */

/* 文件树 */
--file-tree-bg: #0A0E17        /* 文件树 - 主背景 */
--file-tree-selected: #1F2937  /* 选中 - 深灰 */
--file-tree-hover: #111827     /* 悬停 - 次级背景 */
```

## 设计特点

### 1. 去除 AI 味 🚫
- ❌ 无紫色渐变
- ❌ 无粉色、荧光色
- ❌ 无过度鲜艳的糖果色
- ✅ 经典的蓝灰配色
- ✅ 专业的开发工具风格

### 2. 高效开发 💻
- **高对比度** - 代码清晰易读
- **低眼疲劳** - 深色背景护眼
- **层次分明** - 三级背景清晰区分
- **信息突出** - 蓝色主色引导视线

### 3. 视觉平衡 👁️
- **不刺眼** - 避免纯白/纯黑
- **不压抑** - 适度的明暗对比
- **不单调** - 三级灰度层次
- **不花哨** - 克制的色彩使用

### 4. 专业感 🎯
- **蓝色主色** - 可信赖、专业
- **绿色成功** - 积极反馈
- **中性灰调** - 专注内容
- **一致间距** - 4px 基准

## 使用指南

### 切换主题
1. 点击右上角主题切换按钮
2. 选择 "Professional"
3. 应用自动保存偏好

### 快捷键
- `⌘T` - 循环切换主题
- Professional 已设为**默认主题**
